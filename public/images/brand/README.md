# Brand images

Upload here:
- **Logo** — SVG preferred, or high-res PNG with transparent background
- **Favicon** — square PNG (at least 512×512) or `.ico`
- **Social share image (og-image)** — 1200×630 px, used when the site is shared on social/WhatsApp
- (Optional) custom font files in a `fonts/` subfolder

Use simple lowercase names, e.g. `logo.svg`, `favicon.png`, `og-image.jpg`.
