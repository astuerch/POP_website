# Gallery — Event 01 (~50 photos)

**This is where the ~50 photos from the first event go.**

How to upload a big batch via the browser:
1. Open this folder on GitHub.
2. Click **Add file → Upload files**.
3. Drag all ~50 photos in at once (GitHub handles ~100 files per batch, up to 25 MB each).
4. Add a commit message and click **Commit changes**.

Tips:
- Keep original resolution — Next.js generates optimized thumbnails automatically.
- Use simple lowercase names, ideally zero-padded so they sort correctly: `gallery-01.jpg`, `gallery-02.jpg`, … `gallery-50.jpg`.
- Until you upload these, the gallery page shows placeholders.
